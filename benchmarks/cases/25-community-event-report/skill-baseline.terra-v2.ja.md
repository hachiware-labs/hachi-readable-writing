# 子ども向け古本交換会 活動報告

7月12日、集会所で午前10時から正午まで、子ども向けの古本交換会を開きました。読み終えた本を持ち寄り、1冊につき交換券1枚を渡して、会場の本と交換する形です。

事前申込は18家庭でしたが、当日は27家庭、子ども41人が来場しました。集まった本は186冊にのぼり、幅広い本が並びました。一方で、開始直後は受付が一つしかなかったため、持参本の確認に列ができました。10時20分から机を一台増やし、会場係2人が受付を手伝ったことで、10時35分ごろには列を解消できました。

本を選ぶ様子からは、図鑑と低学年向けの読み物に人気が集中したことが分かりました。反対に、高学年向けの本は多く残りました。また、本を選び終えた子どもたちがその場で読み始める場面も多く、急きょ壁際に椅子を8脚置きました。本を交換するだけでなく、子どもたちが本に親しむ時間にもなったように感じます。

交換券を使い切れなかった子どもには、次回も使えることを案内しました。終了時には73冊が残り、持ち主の同意がある本だけを次回まで保管します。けがや本の紛失はありませんでした。

今回、多くの家庭に参加いただけた一方で、受付の混雑と本の対象年齢の偏りという課題も残りました。次回は秋の開催を予定しており、受付を当初から二つ設けるとともに、持ち寄る本の対象年齢の目安を事前にお知らせします。参加しやすく、子どもが自分に合う本と出会える交換会にしていきます。
