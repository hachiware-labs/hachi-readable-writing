# 子ども向け古本交換会 活動報告

7月12日午前10時から正午まで、集会所で子ども向けの古本交換会を開催しました。子どもたちが読み終えた本を持ち寄り、1冊につき交換券を1枚受け取って、会場に並んだ本と交換する仕組みです。

事前申込は18家庭でしたが、当日は27家庭、子ども41人が来場し、合計186冊の本が集まりました。図鑑と低学年向けの読み物が特に人気を集めました。本を選んだあと、その場で読み始める子どもも多かったため、急きょ壁際に椅子を8脚置き、読書できる場所を設けました。本を交換するだけでなく、新しく出会った本をすぐに楽しむ姿が見られたことは、今回のうれしい成果でした。

一方、予想を上回る来場があり、開始直後は受付が混雑しました。受付が一つしかなく、持参された本の確認に時間がかかったためです。10時20分から受付用の机を一台増やし、会場係2人も受付を手伝ったところ、10時35分ごろには列が解消しました。次回は開始時から受付を二つ設け、来場が重なる時間帯にも対応できるようにします。

本の種類にも偏りがあり、図鑑や低学年向けの本に人気が集中する一方、高学年向けの本は多く残りました。子どもの年齢によって希望する本が異なることを踏まえ、次回は、どの年齢層に向いた本を歓迎するか、事前のお知らせで目安を示す予定です。

交換券を使い切れなかった子どもには、残った券を次回も使えると案内しました。終了時に残った本は73冊で、持ち主の同意がある本に限って次回まで保管します。なお、会場内でのけがや本の紛失はありませんでした。

次回は秋の開催を予定しています。今回の混雑や本の偏りを改善しつつ、子どもたちが本を持ち寄り、新しい一冊と出会える機会を引き続きつくっていきます。ご参加、ご協力いただいた皆さま、ありがとうございました。
