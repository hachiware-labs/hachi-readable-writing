# 子ども向け古本交換会 活動報告

7月12日、集会所で午前10時から正午まで、子ども向けの古本交換会を開催しました。事前申込は18家庭でしたが、当日は27家庭、子ども41人が来場し、186冊の本が集まりました。

交換会では、子どもが読み終えた本1冊につき交換券1枚を渡し、その券で会場の本を選ぶ仕組みにしました。多くの子どもが本を選んだあと、その場ですぐに読み始めていたため、急きょ壁際に椅子を8脚置きました。会場で新しい本を手に取り、夢中になって読む姿が多く見られました。

一方、開始直後は受付が一つしかなく、持参された本の確認に列ができました。10時20分から机を一台増やし、会場係2人が受付を手伝ったところ、10時35分ごろには列が解消しました。次回は、最初から受付を二つ設ける必要があります。

本の種類では、図鑑と低学年向けの読み物に人気が集中しました。一方で、高学年向けの本は多く残りました。集まった186冊のうち、終了時点で73冊が残っています。交換券を使い切れなかった子どもには、次回も使えることを案内しました。残った本については、持ち主の同意があるものだけを次回まで保管します。

今回は、けがや本の紛失はありませんでした。参加者が予想を上回ったことで、子どもたちの読書への関心と、このような交流の場への需要を感じる機会になりました。その一方で、受付の混雑や本の種類の偏りも明らかになりました。

次回は秋の開催を予定しています。受付を最初から二つに分けるほか、事前に対象年齢の目安を知らせ、高学年向けの本も集まりやすくする方法を検討します。参加できなかった方も、次回はぜひ読み終えた本をお持ち寄りください。
