# 子ども向け古本交換会 活動報告

7月12日（日）午前10時から正午まで、集会所で子ども向け古本交換会を開催しました。子どもたちが読み終えた本を持参し、1冊につき交換券1枚を受け取り、会場内の本と交換する形式です。

事前申込は18家庭でしたが、当日は27家庭、子ども41人が来場しました。集まった本は186冊で、図鑑や低学年向けの読み物には特に人気が集まりました。一方で、高学年向けの本は比較的多く残りました。終了時の残本は73冊で、このうち持ち主の同意がある本だけを次回まで保管します。

開始直後は受付が一か所しかなく、持参本の確認に時間がかかって列ができました。10時20分から机を1台増やし、会場係2人が受付を手伝ったところ、10時35分ごろには列が解消しました。急な来場増にもかかわらず、けがや本の紛失はありませんでした。

本を選んだ後、その場で読み始める子どもも多く見られたため、壁際に椅子8脚を急きょ用意しました。本を交換するだけでなく、子どもたちが気になる本を手に取り、読書を楽しむ場にもなったことが印象に残りました。交換券を使い切れなかった子どもには、次回も使えることを案内しています。

次回は秋の開催を予定しています。受付は開始時から2か所にし、混雑を減らします。また、図鑑や低学年向けの本に希望が集中したことを踏まえ、持参を呼びかける際には対象年齢の目安をあらかじめお知らせする予定です。今回の反省を生かし、より参加しやすく、希望する本に出会いやすい交換会にしていきます。
