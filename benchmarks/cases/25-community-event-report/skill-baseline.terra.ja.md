# 子ども向け古本交換会 活動報告

7月12日、集会所で午前10時から正午まで、子ども向け古本交換会を開催しました。読み終えた本を持ち寄り、1冊につき交換券1枚をお渡しして、会場の本と交換する形です。

事前申込は18家庭でしたが、当日は27家庭、子ども41人が来場しました。集まった本は186冊となり、幅広い本が並びました。図鑑と低学年向けの読み物には特に人気が集まる一方、高学年向けの本は比較的多く残りました。年齢によって選ばれる本に偏りがあることが分かり、次回に向けた大切な気づきとなりました。

開始直後は受付が一つしかなく、持参本の確認に時間がかかって列ができました。10時20分から机を一台増やし、会場係2人が受付を手伝ったことで、10時35分ごろには列を解消できました。来場が見込まれる時間帯への備えが十分ではなかったため、次回は最初から受付を二つ設けます。

本を選んだあと、会場でさっそく読み始める子どもも多く見られました。そこで急きょ壁際に椅子を8脚置くと、子どもたちが落ち着いて本を開ける場になりました。本を交換するだけでなく、新しい一冊と出会い、その場で読み始められる時間にもなったことが印象に残っています。

交換券を使い切れなかった子どもには、次回も使えることを案内しました。終了時には73冊が残りましたが、持ち主の同意がある本だけを次回まで保管します。けがや本の紛失はありませんでした。

次回は秋の開催を予定しています。受付を当初から二つにすることに加え、持ち寄る本や選ぶ本の対象年齢の目安を事前にお知らせし、低学年向けの本に集中しすぎないよう準備を進めます。今回の混雑や本の偏りを改善しながら、子どもたちが安心して本を選び、地域で本を次につなげられる交換会にしていきます。
