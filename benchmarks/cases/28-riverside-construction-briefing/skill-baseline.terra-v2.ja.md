# 川沿い遊歩道補修工事に関する住民説明会 原稿

## 1. はじめに

本日は、桜橋から中央公園入口までの川沿い遊歩道で予定している補修工事について、ご説明します。

この遊歩道は、散歩や通学、保育園・児童館への移動など、日常的に多くの方に利用されています。高齢の方、小さなお子さんを連れた方、自転車を利用される方にとっては、工事期間中の通行止めや迂回が大きな負担になることを承知しています。

そのうえで、これからも安心して使い続けられるよう、傷みが進む前に補修を行います。本日は、工事の必要性、生活への影響、安全対策、迂回方法、連絡方法の順にお伝えします。

## 2. 工事が必要な理由

7月に遊歩道を点検したところ、舗装のひび割れを18か所、排水がうまく流れない箇所を4か所確認しました。

ひび割れは、雨水が入り込むと傷みが広がり、段差や陥没につながるおそれがあります。また、排水不良が続くと、雨の後に水たまりが残りやすくなり、滑りやすさや通行のしにくさにつながります。

今回の工事では、傷んだ舗装を補修するとともに、排水設備を直します。日々の利用をより安全にし、将来、より大きな工事が必要になることを防ぐための補修です。

## 3. 工事の期間と対象区間

工事期間は、2026年9月1日から11月20日までを予定しています。

対象となるのは、桜橋から中央公園入口までの約600メートルです。この区間は、工事中、終日通行止めとなります。

通行止めによってご不便をおかけしますが、作業中の重機や資材の近くを歩行者や自転車が通ることは危険です。作業場所を明確に分け、安全を確保するため、対象区間全体を通行止めにします。

中央公園と川辺児童館は、工事期間中も通常どおり利用できます。ただし、遊歩道側の入口は閉鎖しますので、市道側など、遊歩道以外からお入りください。

## 4. 生活への影響と騒音の予定

工事車両の出入りは、平日の午前8時30分から午後5時までです。土曜日、日曜日、祝日は、原則として作業を行いません。

また、通学時間帯への影響をできるだけ抑えるため、午前8時から9時までと、午後3時から4時までの間は、大型車両を出入りさせません。

特に音が大きくなる舗装撤去作業は、9月7日から18日までの平日、午前9時から午後4時に予定しています。この期間は、撤去作業に伴う騒音や振動が生じます。通行止めだけでなく、近隣で過ごされる方にもご迷惑をおかけしますが、作業時間を限定して進めます。

天候や現場の状況によっては、工程が変わる場合があります。その際は、後ほどご案内する方法でお知らせします。

## 5. 安全対策と迂回方法

工事中は、対象区間の出入口に通行止めと迂回の案内を掲示します。工事車両が出入りする時間帯には、周辺の安全確認を行いながら作業を進めます。

迂回は市道を通る経路です。通常の遊歩道を通る場合より、約450メートル長くなります。歩く距離が増えるため、特に高齢の方や小さなお子さんと移動される方には、時間に余裕を持っていただくようお願いします。

自転車を利用される方は、迂回路では自転車を降り、押して通行してください。歩行者との接触を防ぎ、狭い場所でも安全に通っていただくためです。

ご不便をお願いする期間ではありますが、通行止め区間へ入ることや、柵を越えて通ることは大変危険です。案内に従い、迂回路をご利用ください。

今回の工事は、約3か月の間、皆さまに迂回や騒音などのご負担をお願いするものです。一方で、ひび割れや排水不良をそのままにせず、今後も安全に利用できる遊歩道を保つために必要な工事です。
