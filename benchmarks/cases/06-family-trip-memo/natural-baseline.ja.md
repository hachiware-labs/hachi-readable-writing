## 金沢1泊2日 家族旅行案

移動を詰め込みすぎず、**1日目は市場と工芸、2日目は庭園**を中心にします。徒歩は1回15分程度を目安にし、観光地間は市内交通を利用。20～30分ごとに座れる休憩を挟む想定です。

### 1日目（土）近江町市場・金箔体験

| 時刻 | 予定 | 負担を減らす工夫 |
|---|---|---|
| 10:30 | 金沢駅着 | 駅でトイレを済ませ、15分ほど休憩 |
| 11:00ごろ | 荷物をホテルへ預ける | 預かり可能か事前確認 |
| 11:30ごろ | 市内交通で近江町市場へ | 徒歩移動を最小限にする |
| 12:00～13:15 | 海鮮の昼食 | 混雑時は行列を避け、座って待てる店や市場周辺の店を選ぶ |
| 13:15～13:45 | 喫茶・休憩 | 市場を歩き続けない |
| 14:00～14:45 | 金箔体験 | 座って楽しめる内容を優先 |
| 15:00すぎ | ホテルへ戻り、チェックイン | 部屋で1時間ほど休む |
| 17:00～18:00 | 金沢駅近くを軽く散策、または自由時間 | 疲れていれば外出しない |
| 18:00～19:30 | 駅近くで夕食 | 海鮮は昼と重なるため、郷土料理なども候補 |
| 19:30以降 | ホテルへ | 翌日に備えて早めに休む |

金箔体験の営業時間・料金・予約可否は不明です。希望する場合は、ホテルから移動しやすく、椅子に座って体験できる施設を事前に確認します。

21世紀美術館は「時間と体力が残っていれば」に留めます。今回は、歩く範囲が広がりやすい**ひがし茶屋街を基本案から外す**のが安全です。

## 雨天時の代案

- 1日目：近江町市場での滞在を短くし、金箔体験とホテル休憩を中心にする。
- 2日目：強い雨なら兼六園を取りやめ、21世紀美術館など屋内候補を中心にする。
- 雨が弱ければ、兼六園は短時間だけ見学し、早めに屋内へ移る。
- 折り畳み傘より、両手が空く雨具や滑りにくい靴を用意する。
- 屋内施設でも入場待ちが長ければ、無理をせず駅周辺で食事・喫茶・買い物に切り替える。

## 事前に確認すること

- ホテルのチェックイン前・チェックアウト後の荷物預かり
- 金箔体験の営業時間、所要時間、料金、予約可否、着席して参加できるか
- 兼六園と21世紀美術館の営業時間、料金、休館・入場制限
- 観光地間の市内交通、乗降場所、混雑見込み
- 食事場所の営業時間、予約可否、子供向け料理、椅子席の有無
- 兼六園で歩きやすい入口や短い見学経路、休憩場所

営業時間や料金が合わなかった場合でも、**近江町市場・金箔体験・兼六園のうち二つを楽しめれば十分**という前提にしておくと、全員が無理なく過ごせます。
