# 金沢1泊2日旅行計画（家族相談用）

祖父母と子供が無理なく過ごせるよう、移動先を絞り、20分以上続けて歩かないことを基本にします。候補のうち、海鮮は近江町市場、工芸は金箔体験、庭園は兼六園を中心にします。ひがし茶屋街は、今回は歩行負担を考えて主な行程から外します。

なお、営業時間・料金・予約の可否は未確認です。以下の時刻は目安として、実施できるかを事前に確認します。

## 1日目：海鮮と金箔体験

| 時刻 | 内容 |
|---|---|
| 10:30 | 金沢駅着。駅でトイレ休憩、荷物の整理 |
| 11:00ごろ | 市内交通で近江町市場へ移動 |
| 11:30ごろ | 近江町市場で海鮮の昼食。待ち時間が長ければ、無理に並ばず別の店を選ぶ |
| 昼食後 | 市内交通でホテル方面へ移動。チェックインまで休憩できる場所を確保する |
| 15:00 | ホテルにチェックインし、全員で休憩 |
| 16:00ごろ | 金箔体験。予約や当日の受付状況を確認できた場合に実施 |
| 夕方以降 | ホテル近くで夕食。移動を短くする |
| 夜 | 翌日の荷物と体調を確認して早めに休む |

金箔体験を実施できない場合は、無理に別の予定を詰め込まず、ホテルで休憩するか、金沢21世紀美術館を代案にします。ただし、美術館の営業時間・料金・混雑状況は未確認です。

## 2日目：庭園を短時間で楽しみ、駅へ戻る

ホテルのチェックアウト時刻と、チェックアウト後に荷物を預けられるかを事前に確認します。荷物を預けられる場合は、身軽になって行動します。

### 晴天時の案

| 時刻 | 内容 |
|---|---|
| 朝 | 朝食、身支度、チェックアウト |
| 午前 | 市内交通で兼六園へ移動 |
| 到着後 | 園内を短い区間ごとに見学し、15～20分歩くごとに休憩 |
| 昼ごろ | 休憩を兼ねて昼食 |
| 午後 | ホテルで荷物を受け取る。必要に応じて駅周辺で休憩や買い物 |
| 15:30ごろ | 金沢駅へ移動し、列車に乗る準備 |
| 16:30 | 帰りの列車に乗車 |

兼六園は全体を見て回ろうとせず、庭園の中で見たい場所をいくつか選ぶ形にします。現地で休憩場所を確認し、祖父の体調に合わせて早めに切り上げます。

### 雨天時の案

雨が強い場合は兼六園を見送り、屋内で過ごせる金沢21世紀美術館を候補にします。

| 時刻 | 内容 |
|---|---|
| 朝 | 朝食、身支度、チェックアウト |
| 午前 | 荷物を預けられるか確認し、市内交通で金沢21世紀美術館へ移動 |
| 午前～昼 | 館内を無理のない範囲で見学。待ち時間が長ければ滞在を短くする |
| 昼ごろ | 昼食と休憩 |
| 午後 | ホテルで荷物を受け取り、金沢駅へ移動 |
| 15:30ごろ | 駅で休憩し、乗車準備 |
| 16:30 | 帰りの列車に乗車 |

雨天時に金沢21世紀美術館を利用できない場合は、観光先を増やさず、ホテルや駅周辺で休憩を多めに取る案に切り替えます。

## 歩行と休憩のルール

- 徒歩だけで移動をつなげず、区間ごとに市内交通を使う。
- 祖父は20分以上続けて歩かない。15～20分を目安に休憩する。
- 祖母、子供も疲れを感じた時点で休む。
- 待ち時間が長い場合は、行列に並び続けない。
- 予定を一つ減らしても、全員が落ち着いて過ごせることを優先する。
- 帰りの列車に遅れないよう、午後3時30分ごろには金沢駅に戻る。

## 予算の考え方

使える金額は、家族全体で70,000円です。料金が未確認のため、次のような上限を決めてから予約・注文します。

| 用途 | 家族全体の目安 |
|---|---:|
| 食事・飲み物 | 35,000円以内 |
| 金箔体験 | 15,000円以内 |
| 金沢21世紀美術館などの施設利用 | 10,000円以内 |
| 市内交通 | 7,000円以内 |
| 予備費 | 3,000円 |
| 合計 | 70,000円以内 |

金箔体験と金沢21世紀美術館の料金が上限を超える場合は、両方を利用せず、金箔体験を優先します。ホテルの朝食が別料金の場合は、食事の予算から支出し、ほかの体験や買い物を減らします。

## 旅行前に家族で確認すること

1. 金箔体験の料金、所要時間、予約の要否。
2. 金沢21世紀美術館の営業状況、料金、雨天時の待ち時間。
3. 近江町市場での昼食候補と、混雑時の代案。
4. ホテルのチェックアウト時刻と、チェックアウト後の荷物預かり。
5. 各区間で利用する市内交通と、おおよその料金。
6. 祖父母と子供が食べやすい夕食の候補。
7. 兼六園を短時間で見学する場合、どこまで見るか。

今回の旅行では、観光地をすべて回ることよりも、近江町市場での食事、金箔体験、兼六園または金沢21世紀美術館のうち、体調と天候に合うものを楽しむことを優先します。
