# 金沢 1泊2日 家族旅行案（2026年8月8日・9日）

祖父が20分以上続けて歩くのが難しいため、宿を金沢駅近くの休憩拠点にし、市内交通を使って移動します。1日に大きな観光は一つを中心にし、食事と休憩の時間を先に確保します。

## 1日目：到着後は屋内中心で、早めに宿へ

10時30分に金沢駅へ到着後、まず荷物を宿に預けます。昼食は近江町市場を第一候補にして海鮮を楽しみますが、混雑していた場合は待ち時間を優先せず、近くで入れる店へ切り替えます。

午後は金沢21世紀美術館を中心にします。屋内で過ごせるため、雨でも予定を大きく変えずに済みます。館内では急がず、座れる場所で一度休憩を入れます。

15時にホテルへチェックインし、祖父母と子供がしっかり休める時間を取ります。夕方以降は、全員の体調がよければ金箔体験を行います。疲れが見られる場合は、体験を翌日に回すか見送り、駅周辺で早めの夕食にします。

## 2日目：兼六園を短く楽しみ、帰路に余裕を残す

朝は慌てずに出発し、兼六園を訪れます。園内を一周することは目指さず、入口から近い範囲を10分ほど歩いたら休憩し、体調を見てもう少しだけ見て回る形にします。庭園を楽しむことを優先し、見られない場所を残しても無理をしません。

昼食後、元気があればひがし茶屋街を短時間訪れます。ただし、歩行や暑さの負担が大きそうなら、金箔体験または金沢21世紀美術館の再訪に切り替えます。午後は早めに金沢駅近くへ戻り、荷物を受け取って休憩します。16時30分の列車に合わせ、駅には余裕を持って戻ります。

## 雨天時の考え方

雨の日は、兼六園やひがし茶屋街を無理に入れず、金沢21世紀美術館と金箔体験を中心にします。近江町市場での食事も含めると、屋内で過ごす時間を増やせます。晴れ間が出た場合だけ、祖父の体調を確認して短時間の庭園散策を加えます。

## 予算の目安

ホテル代と金沢までの交通費を除く7万円は、次の上限で考えます。

| 使い道 | 目安 |
|---|---:|
| 食事・飲み物 | 40,000円 |
| 金箔体験などの体験 | 15,000円 |
| 市内交通 | 10,000円 |
| 混雑時の代替や休憩用 | 5,000円 |
| 合計 | 70,000円 |

金箔体験の料金や予約の要否、各施設・店舗の営業時間は未確認です。出発前に確認し、金箔体験が難しい場合は、その分を食事や休憩の予算に回します。
