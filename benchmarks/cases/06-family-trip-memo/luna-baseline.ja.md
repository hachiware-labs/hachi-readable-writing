## 金沢1泊2日旅行案

「移動を短く、1回の観光は60〜90分程度、必ず座って休む」を基本にします。候補をすべて入れず、庭園・海鮮・工芸を優先します。

### 1日目：近江町市場と金箔体験

| 時刻 | 内容 |
|---|---|
| 10:30 | 金沢駅着。駅でトイレ・休憩 |
| 11:00頃 | 公共交通またはタクシーで近江町市場へ |
| 11:30〜13:00 | 市場で海鮮の昼食。店内で座って休憩 |
| 13:00〜14:00 | 市場内を短時間だけ見学。祖父は無理せず休憩 |
| 14:00頃 | 宿へ移動 |
| 15:00 | チェックイン、部屋で休憩 |
| 16:00〜17:30 | 金箔体験。予約可能か、所要時間、対象年齢を事前確認 |
| 18:00頃 | 金沢駅周辺で夕食 |
| 夜 | 宿で休む。翌日に備えて外出は控える |

金箔体験が予約できない場合や、1日目の疲れが強い場合は、体験を取りやめて宿で休みます。

### 2日目：兼六園を中心にする日

| 時刻 | 内容 |
|---|---|
| 朝 | 朝食後、ゆっくり出発 |
| 9:30頃 | 公共交通またはタクシーで兼六園方面へ |
| 10:00〜11:30 | 兼六園を短い範囲に絞って見学。20分歩く前に必ず休憩 |
| 11:30〜12:30 | 昼食・休憩 |
| 13:00〜14:30 | 体力に応じて、次のどちらかを選択 |
|  | A：金沢21世紀美術館 |
|  | B：宿または駅周辺へ戻って休憩 |
| 14:30〜15:30 | 金沢駅へ移動、買い物・休憩 |
| 16:30 | 帰りの列車 |

ひがし茶屋街は、街歩きと移動が増える可能性があるため、今回は基本的に外します。家族全員の体力に余裕がある場合だけ、2日目の兼六園または美術館の代わりに検討します。

## 雨天時の代案

- 1日目：近江町市場で昼食 → 宿で休憩 → 金箔体験
- 2日目：兼六園を取りやめ、金沢21世紀美術館を中心にする
- 美術館の待ち時間や入場可否が不明な場合は、金沢駅周辺で昼食・買い物・休憩に変更
- ひがし茶屋街は雨天時には無理に入れない

## 予算の目安

7万円を超えないよう、事前に次の上限を決めます。

- 食事・飲み物・間食：4万円
- 金箔体験など：1万5千円
- 市内交通：1万5千円

金箔体験の料金、各施設の入場料、営業時間、予約の可否は未確認のため、予約・購入前に確認が必要です。特に、金箔体験は5人分の料金とキャンセル条件を確認します。

## 家族で決めておくこと

- 20分以上連続して歩かない
- 観光地ごとに休憩場所を先に確認する
- 2日目午後は「美術館へ行く」か「駅周辺で休む」か、当日の体力で決める
- 雨の場合は兼六園とひがし茶屋街を優先的に省略する
- 移動は公共交通を基本にし、歩行負担が大きい区間はタクシー利用も予算内で検討する
