# 金沢1泊2日 家族旅行案

祖父母と子供の負担を抑え、庭園・工芸・海鮮を一つずつ楽しむ計画です。移動を詰め込まず、歩行は20分未満を目安に区切ります。

## 基本方針

- 主な目的地は、兼六園・金箔体験・近江町市場の3か所とする。
- ひがし茶屋街と金沢21世紀美術館は、体調や天候に応じて選ぶ候補とする。
- 午前と午後にそれぞれ休憩を取り、昼食にも十分な時間を確保する。
- 長い徒歩移動を避け、市内交通を利用する。乗り場や移動時間は事前に確認する。
- 混雑や待ち時間が長い場合は、予定を一つ減らしてホテルや駅周辺で休む。

## 1日目：工芸を楽しみ、早めにホテルへ

**10:30　金沢駅到着**

荷物を預けられるかホテルに確認し、可能なら先に預けます。到着直後は移動を急がず、駅周辺で休憩します。

**11:15ごろ　昼食**

駅周辺で海鮮を含む昼食を取ります。近江町市場での食事は翌日に予定しているため、ここでは待ち時間の短さを優先します。

**12:45ごろ　金箔体験**

工芸の希望を満たす、この日の中心予定です。場所、所要時間、料金、6歳の子供が参加できるか、座って体験できるか、予約の要否は事前確認が必要です。

**14:15ごろ　休憩・ホテルへ移動**

体験後は予定を追加せず、休憩を挟んでホテルへ向かいます。

**15:00　チェックイン**

部屋で1時間ほど休みます。祖父母だけでなく、子供の疲れもここで整えます。

**16:30ごろ　体調に応じた短い外出**

全員に余裕があれば、ひがし茶屋街を短時間訪れます。歩行や混雑の負担が大きそうなら行かず、駅周辺で過ごします。茶屋街では広く回ろうとせず、休憩を含めて早めに切り上げます。

**18:00ごろ　夕食**

ホテルまたは駅の近くで取り、食後は予定を入れません。

## 2日目：兼六園と近江町市場

**8:00ごろ　朝食・出発準備**

チェックアウト後の荷物をホテルで預けられるか、あらかじめ確認します。

**9:30ごろ　兼六園**

庭園を楽しむ、この日の中心予定です。園内をすべて回ることは目指さず、歩行を15～20分以内に区切って休憩します。入口や園内の段差、休憩場所、市内交通の乗降場所は事前に確認しておきます。

**11:00ごろ　休憩・移動**

祖父の疲れが見える場合は、この後の予定を近江町市場だけに絞ります。

**11:45ごろ　近江町市場で昼食**

海鮮を楽しみます。混雑している場合は、長時間並ばず、入れる店や周辺の飲食店へ切り替えます。

**13:15ごろ　午後の選択**

当日の体調と天候に合わせ、次のどちらかを選びます。

- 余裕がある場合：金沢21世紀美術館を短時間見学する
- 疲れがある場合：金沢駅へ戻り、座って休憩や買い物をする

美術館の営業時間、料金、混雑状況、休館の有無は未確認です。訪問候補にする場合は事前に確認します。

**15:00ごろ　金沢駅へ**

ホテルで荷物を受け取り、遅くとも出発の1時間前を目安に駅へ戻ります。

**16:30　帰りの列車に乗車**

## 雨天時の変更案

小雨で祖父母の足元に問題がなければ、兼六園は滞在を短くして予定どおり訪れます。強い雨や足元への不安がある場合は兼六園とひがし茶屋街を外し、金箔体験と金沢21世紀美術館を中心にします。

ただし、屋内施設でも入口までの移動や館内歩行があります。雨天時は目的地を増やさず、ホテルや駅周辺で休む時間を長めに取ります。

## 予算の目安

7万円の範囲で調整するための仮配分です。実際の料金は未確認のため、予約前に見直します。

| 用途 | 家族5人分の目安 |
|---|---:|
| 1日目の昼食 | 10,000円 |
| 1日目の夕食 | 15,000円 |
| 2日目の昼食 | 12,000円 |
| 飲み物・休憩 | 5,000円 |
| 金箔体験・施設料金 | 13,000円 |
| 市内交通 | 10,000円 |
| 予備費 | 5,000円 |
| **合計** | **70,000円** |

朝食がホテル代に含まれない場合は、食費または予備費の配分を変更します。体験料や市内交通費が想定を上回る場合は、ひがし茶屋街や美術館を減らし、予算と休憩時間を優先します。

## 事前に確認すること

- ホテルで、到着前とチェックアウト後に荷物を預けられるか
- 各目的地への市内交通、乗り場、乗り換え、徒歩時間
- 兼六園の営業時間、料金、入口、段差、休憩場所
- 金箔体験の場所、料金、所要時間、予約の要否、年齢条件、座って参加できるか
- 金沢21世紀美術館の営業時間、料金、休館情報
- 近江町市場で昼食を取れる時間帯と、混雑時の代替店
- ホテルの朝食条件
- 当日の天気予報

この計画では、金箔体験と兼六園を優先し、海鮮は駅周辺と近江町市場で楽しみます。ひがし茶屋街と美術館は必須にせず、全員の体調がよければ追加する位置づけにします。
