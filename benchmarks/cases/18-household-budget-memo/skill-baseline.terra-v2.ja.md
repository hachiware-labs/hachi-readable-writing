# 冷蔵庫の修理・買い替え相談メモ

家電用積立は220,000円ですが、翌月の学校関係の支払い80,000円を同じ積立から出すため、冷蔵庫に使える金額は実質140,000円です。借り入れや分割払いは使わない方針です。

現時点では冷却できているため、家計だけで判断すると修理を選ぶのが現実的です。修理費38,000円なら、学校関係の支払い後にも102,000円を残せます。一方、買い替えは配送・回収込みで148,000円のため、学校関係の支払いを考慮すると積立だけでは8,000円足りません。先に買い替えた場合でも、積立の残りは72,000円となり、翌月の支払い80,000円をまかなえません。

ただし、修理箇所の保証は90日で、その後どのくらい使えるかは分かりません。作動音が断続的に大きい状態も続いているため、修理は「当面使い続けるための支出」と考える必要があります。新製品は5年保証があり、販売店の試算では電気代が月1,500円安くなる見込みです。ただし、この節約額は保証されたものではありません。仮に試算どおりでも、購入額と修理額の差額110,000円を電気代だけで埋めるには約6年1か月かかります。

したがって、今回は38,000円で修理し、残る積立102,000円を次の買い替え資金として確保する案を基本にしたいです。その後も異音や冷却の状態を見て、積立を補充しながら買い替え時期を準備します。

ただし、冷えなくなった場合は修理より買い替えを優先します。その時点で積立だけでは不足するため、借り入れや分割払いを使わずにどう資金を確保するかは、別途家計全体で相談が必要です。
