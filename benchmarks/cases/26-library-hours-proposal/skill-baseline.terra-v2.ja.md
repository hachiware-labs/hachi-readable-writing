# 火曜・木曜の閉館時間延長試行について

本日は、火曜と木曜に限り閉館時間を午後8時まで延長する3か月間の試行について、ご承認をお願いします。あわせて、警備費と光熱費として必要な42万円の予算承認をお願いしたいと考えています。

現在、図書館は火曜から日曜まで、午前9時から午後6時まで開館しています。この時間は多くの利用者に利用されていますが、平日の夕方以降に利用したい方には十分とはいえません。

3月に利用者320人を対象に実施したアンケートでは、回答者の62％が平日午後7時以降の利用を希望しました。特に希望が多かったのは、仕事を終えてから来館する会社員と、放課後に学習場所を必要とする高校生です。

実際の利用状況も、この希望を裏づけています。4月の平日午後5時から6時までの来館者数は、1時間平均84人でした。これは日中の平均46人を大きく上回ります。閉館前の1時間に利用が集中していることから、午後6時以降にも一定の利用需要があると考えられます。

一方で、毎日閉館時間を延長するだけの人員はありません。そこで、需要が見込まれる平日のうち、火曜と木曜だけを午後8時まで開館する案を提案します。期間は6月から8月までの3か月間です。まず限定的に実施し、利用状況と運営への影響を確認したうえで、継続の可否を判断します。

午後6時以降は、サービスを1階に絞ります。利用できるのは閲覧、貸出、返却です。児童書エリアと会議室は午後6時で閉鎖します。これにより、夜間に必要な人員と安全管理の範囲を限定しつつ、会社員や高校生が求める基本的な利用機会を確保します。

遅番は職員2人と警備員1人を配置します。職員については既存シフトを組み替えて対応し、残業は月合計24時間以内に収まる見込みです。ただし、試行中は職員の負担も記録し、業務量や安全面に無理がないかを確認します。

追加費用は、警備と光熱費を合わせて3か月で42万円です。毎日延長するのではなく週2日に限定することで、費用を抑えながら利用需要を確かめられます。今回の試行は、恒久的な拡大を先に決めるものではなく、限られた予算で判断材料を得るためのものです。

試行期間中は、午後6時以降の時間帯別来館者数、貸出数、費用、職員の負担を記録します。利用者数だけでなく、貸出や返却の実績、運営に必要な負担をあわせて見て、延長開館が図書館サービスとして成り立つかを判断します。

継続の目安は三つです。第一に、午後6時以降の平均来館者数が1日50人以上であること。第二に、重大な安全上の問題がないこと。第三に、追加費用が予算内であることです。この条件を満たすかを試行終了後に検証し、継続、見直し、終了の判断を会議にお示しします。

火曜と木曜の午後8時までの開館は、毎日の延長が難しいという制約の中で、夕方以降の利用希望に応える現実的な案です。利用者の声と現在の夕方の来館状況を踏まえ、まず3か月間、範囲を限定して確かめたいと考えています。

以上より、6月から8月までの試行実施と、追加費用42万円の予算承認をお願いします。
