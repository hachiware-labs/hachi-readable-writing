# 火曜・木曜の閉館時間延長に関する試行提案

本日は、市立図書館の閉館時間を週2日、火曜と木曜に限って午後8時まで延長する試行について、ご承認をお願いしたく、ご説明します。

現在の開館時間は、火曜から日曜まで午前9時から午後6時です。この時間帯は多くの利用者にとって利用しやすい一方、仕事や学校の後に図書館を利用したい方には、十分とは言えません。

3月に利用者320人を対象として実施したアンケートでは、回答者の62％が、平日午後7時以降の利用を希望しました。特に希望が多かったのは、会社員と高校生です。会社員にとっては退勤後、高校生にとっては部活動や授業の後に立ち寄れる時間が必要です。現在の午後6時閉館では、利用したくても来館できない人が一定数いると考えられます。

実際の利用状況も、この要望と一致しています。4月の平日午後5時から6時までの来館者数は、1時間平均84人でした。これは日中の平均46人を大きく上回ります。閉館前の1時間に利用が集中していることからも、夕方以降に図書館を利用したい需要はすでに表れています。

ただし、毎日閉館時間を延長するだけの人員はありません。そこで、利用者の要望に応えながら、職員の負担と費用を抑える案として、火曜と木曜だけ午後8時まで開館する試行を提案します。

試行期間は、6月から8月までの3か月間です。夏季は学校の課題や調べ学習、仕事帰りの利用なども見込まれます。まずは期間と曜日を限定して実施し、延長開館に継続的な需要があるかを確かめます。

午後6時以降は、利用できる場所と業務を絞ります。児童書エリアと会議室は午後6時で閉鎖し、延長時間帯に利用できるのは、1階の閲覧、貸出、返却です。これにより、限られた人員でも利用者の安全を確保しながら、図書館として中心的なサービスを提供できます。

遅番の体制は、職員2人と警備員1人です。既存シフトを組み替えて対応するため、残業は月合計24時間以内に収まる見込みです。毎日の延長ではなく週2日とすることで、職員の負担を管理可能な範囲にとどめます。

追加費用は、警備と光熱費を合わせて、3か月で42万円を見込んでいます。この費用は、単に開館時間を増やすための支出ではありません。これまで利用の機会を得にくかった会社員や高校生に、図書館への入口を広げるための試行費用です。同時に、今後の開館時間を判断するための実測データを得る費用でもあります。

試行中は、午後6時以降の時間帯別来館者数と貸出数、追加費用、職員の負担を記録します。利用者数だけではなく、実際に貸出や返却につながっているか、運営体制に無理がないかを合わせて確認します。

継続の目安は明確にします。午後6時以降の平均来館者数が1日50人以上であること、重大な安全上の問題がないこと、そして追加費用が予算内であることです。この三つを満たした場合に、試行後の継続を検討します。逆に、利用が見込みを下回る場合や、安全・運営面で課題が大きい場合は、曜日や時間帯を見直す、あるいは延長を終了する判断をします。

今回お願いしたいのは、6月から8月まで、火曜と木曜の閉館時間を午後8時まで延長する試行の実施と、そのための追加予算42万円の承認です。

限られた人員の中で、毎日延長することはできません。しかし、利用が集中する夕方以降に、週2日でも図書館を開けることには意味があります。まずは3か月間、利用実態と運営上の課題を確かめ、今後の開館時間を根拠を持って判断できるようにしたいと考えています。

ご承認をお願いいたします。
