# Terra skill effectiveness evaluation

評価日: 2026-07-12  
対象: `terra-baseline.*.md` と、Terra対応後の `skill-baseline.terra-v2.*.md` の全60組
生成モデル: `gpt-5.6-terra`  
推論設定: `high`

Terraの同一題材について、スキルなしとTerra対応後の `hachi-readable-writing` を比較した。スキルを使ったこと自体は加点せず、人間が流れを追って理解できるかだけを評価している。旧スキルの結果は平均89.1点であり、Terra対応後は92.5点となった。

Terra対応後は25文書が改善、21文書が同点、14文書が悪化した。旧スキルの改善8、同点11、悪化41から大きく改善している。

効果が大きかったのは、言語不一致を修正したケース17・19英語版、入力外の料金やURLを除いたケース06英語版、コード範囲を抑えたケース08日英である。残る課題はプレゼンで、ケース28と30は発表時間を意識して短くしすぎ、説明の厚みを失った。Terra向けの保存優先と局所修正は有効だが、時間制限は上限だけでなく必要な説明量も考慮する必要がある。
